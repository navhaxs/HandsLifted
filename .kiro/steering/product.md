# Product Overview

VisionScreens (codebase name: HandsLifted) is church presentation software designed for modern worship services. It handles song lyrics display, media playback, slide presentations, and live output to multiple screens.

## Key Capabilities

- Song lyrics display with customizable slide themes and arrangements
- PowerPoint/PDF/Google Slides import and rendering
- Video playback via LibMPV and VLC
- NDI (Network Device Interface) output for broadcast workflows
- Stage display / confidence monitor output
- Slide designer for custom slide templates
- Song library management and online lyrics import
- Google Drive integration for cloud-based assets
- OBS integration for streaming setups
- Wii Remote support as a wireless presentation clicker

## Target Users

Church worship teams and AV operators who need to project lyrics, scripture, announcements, and media during services.

## Status

Active development (2025 branch). Under heavy and rapid iteration.
